# DIG 4503c portfolio project

## Description

This project is a personal portfolio design that showcases the projects I’ve created, my skills, an “About Me” section, and a way to contact me.

## Design / Deployment
I design thie portfolio as a react app and deploy the design through vercel.

## Page designs
- Home page: an introduction, greeting message, quick action buttons
- About page: A description about me, my skills, and an image of me
- Projects page: Two of my best projects, project description, tech stack, and deploy links
- Contact page: Input form field, social links, email
- Navbar: Page displays (Home,About,Projects, Contact)
- Footer
